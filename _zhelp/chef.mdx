Chef 是一套自动化运维工具。

### Debian/Ubuntu 用户

再选择你的 Debian/Ubuntu 版本，文本框中内容写进 `/etc/apt/sources.list.d/chef.list`

<CodeBlock
  enableQuickSetup
  filepath="/etc/apt/sources.list.d/chef.list"
  menus={[
    {
      title: 'Ubuntu/Debian 版本',
      items: [
        ['Ubuntu 20.04 LTS', { release_name: 'focal' }],
        ['Ubuntu 18.04 LTS', { release_name: 'bionic' }],
        ['Ubuntu 16.04 LTS', { release_name: 'xenial' }],
        ['Ubuntu 14.04 LTS', { release_name: 'trusty' }],
        ['Debian 11 (bullseye)', { release_name: 'bullseye' }],
        ['Debian 10 (buster)', { release_name: 'buster' }],
        ['Debian 9 (stretch)', { release_name: 'stretch' }],
        ['Debian 8 (jessie)', { release_name: 'jessie' }]
      ]
    }
  ]}
>

```properties
deb {{http_protocol}}{{mirror}}/apt/stable {{release_name}} main
```

</CodeBlock>


### RHEL/CentOS 用户

新建 `/etc/yum.repos.d/chef.repo`，内容为：

<CodeBlock
  enableQuickSetup
  filepath="/etc/yum.repos.d/chef.repo"
>

```ini
[chef-stable]
name=chef-stable
baseurl={{http_protocol}}{{mirror}}/yum/stable/stable-el$releasever-x86_64/
enabled=1
gpgcheck=1
gpgkey=https://packages.chef.io/chef.asc
```

</CodeBlock>
