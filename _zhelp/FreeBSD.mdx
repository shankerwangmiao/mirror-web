- ISO 镜像位于

    <CodeBlock>
    ```
    {{http_protocol}}{{mirror}}/releases/ISO-IMAGES
    ```
    </CodeBlock>

- 虚拟机模板位于

    <CodeBlock>
    ```
    {{http_protocol}}{{mirror}}/releases/VM-IMAGES
    ```
    </CodeBlock>

- 文档及资料位于

    <CodeBlock>
    ```
    {{http_protocol}}{{mirror}}/doc
    ```
    </CodeBlock>

pkg 和 ports 仓库已被官方移动到独立的仓库中。详见 [FreeBSD pkg 帮助](/FreeBSD-pkg/)与 [FreeBSD ports 帮助](/FreeBSD-ports/)
