该仓库包含了所有 Ubuntu 以前发布过的软件仓库、镜像 ISO，但不包含 Ubuntu 衍生版的 ISO。

```properties

deb {{http_protocol}}{{mirror}}/ubuntu/ {{release_name}} main restricted universe multiverse
{{enable_source}}deb-src {{http_protocol}}{{mirror}}/ubuntu/ {{release_name}} main restricted universe multiverse
deb {{http_protocol}}{{mirror}}/ubuntu/ {{release_name}}-updates main restricted universe multiverse
{{enable_source}}deb-src {{http_protocol}}{{mirror}}/ubuntu/ {{release_name}}-updates main restricted universe multiverse

# 预发布软件源，不建议启用
{{enable_proposed}}deb {{http_protocol}}{{mirror}}/ubuntu/ {{release_name}}-proposed main restricted universe multiverse
{{enable_proposed}}{{enable_source}}deb-src {{http_protocol}}{{mirror}}/ubuntu/ {{release_name}}-proposed main restricted universe multiverse
```

### 镜像

请前往

<CodeBlock>
```
{{http_protocol}}{{mirror}}/releases/
```
</CodeBlock>

下载。

非 AMD64(x86_64), Intel x86 架构的镜像请前往

<CodeBlock>
```
{{http_protocol}}{{mirror}}/releases/ports/releases/
```
</CodeBlock>

下载。
